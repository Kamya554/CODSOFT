# 🚢 Titanic Survival Prediction - CODSOFT Internship Task 1

This project is part of the CODSOFT Data Science Internship. The goal is to build a machine learning model that predicts whether a passenger on the Titanic survived based on available data such as age, gender, class, fare, etc.

---

## 📌 Problem Statement

Use the Titanic dataset to train a model that can predict passenger survival. This is a binary classification problem (Survived = 0 or 1).

---

## 📁 Files in This Project

- `train.csv` – Training data with survival labels
- `test.csv` – Test data without labels
- `gender_submission.csv` – Sample submission format
- `titanic_prediction.py` – Python script used to train and evaluate the model
- `my_submission.csv` – Model predictions (if generated)

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- scikit-learn (DecisionTreeClassifier)
- Jupyter / Command Line
